为什么要写博客？

发表于 2019-12-19 |

终于搭建起来了自己的博客，之前一直在CSDN上面写博客，后面发现很多人都是使用Hexo + github + 域名搭建的博客系统，因为自己比较懒，一直没有去做，马上20年了，就花了一段时间自己动手搭建了一下博客，下面写一下我对博客的一些理解

为什么要写博客

为什么要写博客呢？很多人刚开始都不知道为什么要写博客，只写看别人的文章，听别人说要写博客，然后就开始写博客了，其实说实话我也是这样。刚开始在CSDN上面写博客，因为写起来比较简单，不需要自己搭建什么东西。后面看见身边的人博客都是在用自己域名的，觉得逼格一下就高了，然后我也就找了时间，学习一下怎么使用域名搭建自己的博客。

那么现在说说我为什么要写博客

  1. 为了养成一个习惯吧，坚持做一件事情可以让自己养成一个比较好的习惯，但是要想找到让自己坚持并且有意义的事情是比较有难度的，那么博客相对于大多数人来说，会是一件首先想到的事情
  2. 为了方便记录知识，我们经常会遇到解决一个问题之后，因为没有记录到下次再次遇到这个问题的时候不知道如何去解决了，所以博客就是一个很好的记录工具，当然github也是一个很好的记录工具
  3. 为了方便自己找工作，因为一份简历可以体验出来的东西比较少，为了让别人了解你，那么博客也是一个非常好的方式
  4. 为了提升自己，想让别人了解你，提升自己的价值，当我们的博客浏览数一直增加的时候，我们就会收获一种满足感和成就感，这种感觉可以激励我们自己

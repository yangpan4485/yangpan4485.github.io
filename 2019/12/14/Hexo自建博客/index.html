Hexo自建博客

发表于 2019-12-14 | | 浏览

这一篇主要记录一下我是如何使用Hexo+github+域名的方式搭建自己博客的

github新建仓库和git配置

登陆github创建个人仓库

登陆github，没有github账号的需要自己去注册一个，然后新建一个项目(new repository)，命名为：用户名.github.io，比如我的仓库名是：yangpan4485.github.io

安装git

linux和mac使用命令行安装git，Windows需要从git官网下载git，然后安装。如果不熟悉git推荐在网上找一些资料学习一下

git配置

配置user.name和user.mail

生成ssh密钥文件

在.ssh文件夹下面找到id_rsa.pub文件，将其内容全部复制，打开github配置ssh key页面，新建一个New SSH key，将复制的内容添加进去。具体的配置可以参考这篇文章
这个时候你的git就配置完成了，下面使用git clone 就可以拉下你的仓库到本地了

Hexo的配置

安装Node.js和npm

Hexo基于Node.js，linux和mac同样都是使用命令行进行手动安装，Windows需要下载安装，下载Node.js，Windows安装Node.js会包含npm的安装，安装后检测Node.js和npm是否安装好
检查Node.js和npm是否安装成功

安装和配置Hexo

接下来的操作都会以mac电脑为基础
安装Hexo

初始化我们的博客，会在本地创建一个blog文件夹

这个时候会有一篇初始化的博客，然后我们输入hexo s，然后输入http://localhost:4000/，就可以看到画面了

将Hexo和github互相关联起来

首先先确定两个配置文件，一个称为站点配置文件，是我们主目录下面的_config.yml文件，还有一个是主题配置文件，是在/thems/主题名/下面的_config.yml文件
修改站点配置文件(主目录下面的_config.yml文件)，将最后几行代码修改为下面的内容

deploy:
type: git
repository: 之前在github创建的仓库的路径
branch: 分支

例如我的是：

deploy:
type: git
respsitory: https://github.com/yangpan4485/yangpan4485.github.io.git
branch: master

安装git部署插件

接下来执行下面几条命令

接下来我们在浏览器中输入用户名.github.io就可以看见我们之前看到的内容了

配置Hexo主题

Hexo默认的主题是landscape，并不是多么的好看，接下来我们配置我们自己想要的主题，在配置主题之前你可以看一看自己喜欢哪一种主题，Hexo主题在这里面有详细的主题介绍，然后选择自己想要的主题去github上搜索，再将其clone到thems目录下面，我使用的是Next主题
修改站点配置文件

title: xxx
description: xxx
author: xxx
language: zh-Hans
theme: next

修改完主题之后执行hexo clean && hexo g && hexo d，然后在浏览器里面输入用户名.github.io就可以看见我们更改过后的主题了

配置域名

购买域名

我们可以在Godaddy，腾讯云，百度云等其它域名购买方法

域名解析和托管

这里使用DNSPod解析和托管域名，

  • 添加域名
  • 双击域名
  • 添加三条解析记录，将域名与github向关联起来
    添加两条A记录
    * –> 192.30.252.153
    @ –> 192.30.252.153
    添加一条CNAME记录
    www –> xxx.github.io
    如下图所示
  • 修改域名DNS
    使用自定义的DNS服务器，将NDS服务器修改为
    f1g1ns1.dnspod.net
    f1g1ns2.dnspod.net
  • 博客添加CNAME文件
    将域名DNS服务器修改成功后，进入博客目录，在source目录下新建CNAME文件，写入域名，例如：yangpan.online

最后我们输入我们的域名就可以看见我们的博客了
